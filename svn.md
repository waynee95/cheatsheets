# SVN 

## Install

```
sudo apt install subversion
```

## SVN checkout

```
sudo svn checkout --username <name> <url>
```

## SVN update

```
sudo svn update
```

## SVN status

```
sudo svn status
```

## SVN add

```
svn add <file>
```

## SVN revert

```
sudo svn revert <file>
```

## SVN commit

```
svn commit -m "your message"
```
